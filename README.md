# Calculadora de Amor —

This repository hosts a demo page for **Calculadora de Amor**.  
The demo is served with GitLab Pages and links to the full product at:

- Main website: https://calculardeamor.com  
- Full calculator: https://calculardeamor.com/calculadora-de-amor/

## Files
- `.gitlab-ci.yml` — GitLab Pages deployment config
- `public/index.html` — demo page with a responsive love calculator widget and backlinks

## About
Calculadora de Amor is a fun tool for estimating compatibility between two names. This demo page is meant to be indexed by search engines and provide a contextual backlink to the main site.

> Note: The calculator is for entertainment purposes only.
